터미널
gitbash
$ touch 부르고 싶은 html 문서들 -> 지웠던 문서들이 다시 생성됨
$ mkdir 단어 - 단어로 써진 폴더 생성됨
$ git --version -> git 버전이 몇인지 알 수 있음
$ cd .. -> 상위 폴더로 이동
$ rmdir 비어있는 디렉토리 삭제
$ git init 비어있는 숨김폴더 생성
$ ls -al 모든 파일 보기
$ git config --global user.name "jihye.kwon"
$ git config --global user.email kwonjhjyj@naver.com
$ git config --list
$ touch README.md -> readme 파일 생성
